Resource management class template.

Usage:

Constructor & Destructor Documentation

template<typename T >
ogle::ResourceManager< T >::ResourceManager ( const ConstructorTag &  private_tag)
explicit

The constructor.

Since the std::make_unique in ResourceManager<T>::Instance() can't access a private constructor, we make this constructor public but need a ConstructorTag parameter. ConstructorTag is an empty struct only accessible by this class, since it is private. This way the constructor can only actually be used in a private setting.
